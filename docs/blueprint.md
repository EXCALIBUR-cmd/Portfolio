# **App Name**: MotionPort

## Core Features:

- Locomotive Scroll Integration: Smooth scrolling powered by Locomotive Scroll for a unique browsing experience.
- GSAP Animations with AI Assistance: GSAP (GreenSock Animation Platform) for complex, high-performance animations and transitions. Use AI tools to automatically generate suggested animations based on the content displayed.
- Project Showcase: Clean, modern design focusing on showcasing projects effectively.
- Responsive Layout: Responsive design to ensure the portfolio looks great on all devices.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a professional and modern look.
- Secondary color: Light gray (#F0F0F0) for backgrounds and subtle contrast.
- Accent: Teal (#008080) for interactive elements and highlights, providing a touch of vibrancy.
- Clean and readable sans-serif fonts for body text and headings.
- Minimalist icons to represent skills, technologies, and contact information.
- Full-screen sections with smooth transitions between them.
- Subtle animations on hover and scroll to engage the user.
